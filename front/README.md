# EssLearn Front (Angular)

This is an Angular scaffold that mirrors the original `frontend/` design. It's a starting point — styles from the original project have been copied into `src/styles/`.

Quick start:

```bash
cd front
npm install
npx ng serve --open
```

Notes:
- This project is a minimal scaffold. Use the original `frontend/` React app as a visual reference and port components into Angular under `src/app`.
